Video App

Este aplicativo transmite conteúdo de vídeo S3 por HTTP usando a API Amazon S3 Java V2.

  1. Faça upload de vídeos MP4 para um bucket do Amazon S3, escolhendo o video.
  2. Preencha o formulário e envie.
  3. The application stores the video in an Amazon S3 bucket.
  4. Você pode assistir um conteúdo escolhendo umvideo.
  5. Em seguida, escolha o video clicando no título.
